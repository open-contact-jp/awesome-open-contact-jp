### プログラマに準ずる方へ {#for-programmer}

本サイトは[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)で構築されています。  
MkDocsの詳細は以下を参照ください。

* [MkDocsの公式ドキュメント](https://www.mkdocs.org/)
* [Material for MkDocsの公式ドキュメント](https://squidfunk.github.io/mkdocs-material/)

[Python](https://www.python.org/)環境が必要なので以下等を参照してインストールください。

* [PyEnv](https://github.com/pyenv/pyenv)
* [MISE](https://mise.jdx.dev/lang/python.html)
* [Python公式](https://www.python.org/)

その後[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)をインストールし、

```
pip install mkdocs-material
```

カレントディレクトリを[GitHub | awesome-open-contact-jp](https://github.com/open-contact-jp/awesome-open-contact-jp)
をクローンした箇所に移動させて、  
以下のコマンドでローカル確認用のサーバー立ち上がります。

```
mkdocs serve
```

修正後は以下コマンドでsiteディレクトリを更新して、
そのソースも含めてコミットください。

```
mkdocs build
```

※興味はあるけれど上記の意味がわからないという方は、  
[Discordコミュニティ(無料)](#discord-community)より`#awesome-open-contact-jp`または管理者(`Sean Daru`)に直接ご連絡ください。
